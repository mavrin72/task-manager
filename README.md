# Task Manager

React + TypeScript додаток для управління задачами з Telegram ботом.

## Веб-додаток

```bash
npm install
npm run dev
```

Відкрий [http://localhost:5173](http://localhost:5173).

## Telegram бот

Бот дозволяє керувати задачами прямо з Telegram: переглядати, додавати, позначати виконаними та видаляти задачі.

### Налаштування

**1. Створи бота в Telegram**

Відкрий [@BotFather](https://t.me/BotFather), відправ `/newbot`, дай ім'я та юзернейм (має закінчуватись на `bot`). Скопіюй токен.

**2. Встанови залежності**

```bash
cd bot
npm install
```

**3. Налаштуй `.env`**

```bash
cp .env.example .env
```

Відкрий `bot/.env` і встав свій токен:

```
TELEGRAM_BOT_TOKEN=1234567890:AAF...
```

Опційно — обмеж доступ лише для своїх акаунтів (знайди свій ID через [@userinfobot](https://t.me/userinfobot)):

```
ALLOWED_USER_IDS=123456789,987654321
```

**4. Запусти бота**

```bash
npm run dev
```

### Команди бота

| Команда | Опис |
|---------|------|
| `/tasks` | Список активних задач |
| `/all` | Всі задачі (активні + виконані) |
| `/add <назва>` | Додати нову задачу |
| `/done <номер>` | Позначити виконаною |
| `/undone <номер>` | Зняти позначку виконання |
| `/delete <номер>` | Видалити задачу |
| `/info <номер>` | Детальна інформація про задачу |
| `/help` | Довідка |

### Додаткові параметри для /add

```
/add Зробити звіт --priority high --initiator Іван --deadline 2025-12-31
```

- `--priority low|medium|high` — пріоритет (за замовчуванням: medium)
- `--initiator <ім'я>` — відповідальний
- `--deadline YYYY-MM-DD` — дедлайн

### Синхронізація з веб-додатком

Бот та веб-додаток зберігають задачі в **різних сховищах**:
- Веб-додаток використовує `localStorage` браузера
- Бот використовує файл `tasks.json` у кореневій директорії проекту

Для синхронізації можна:
- Експортувати задачі з браузера (DevTools → Application → localStorage → `task-manager-tasks`) і помістити у `tasks.json`
- Або налаштувати спільний backend (наступний крок розробки)
